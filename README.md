数据结构学习资料库

这里存放的是我的数据结构课程 PDF 讲义和相关笔记。

数据结构是一门内容量非常大的课程，但胜在老师非常的友好，自己复习的也是比较认真，给了自己几点教训吧。

一是这种难的课，学期中一定要好好学一学，不然真的很难很难复习，看不懂会导致拖慢复习进度，极大地拖慢，对心理上也很难受，对这么课的老师也是，一开始一直处于一个抗拒状态，觉得讲的不好不想听不愿学，实则只是自己无法集中注意力，要么课上认真听，要么课后自己去钻研，出来混总是要还的。

二是代码能力上，虽然进度很慢，但自己动手去写，真的可以看见非常明显的能力的提升，真的很不错的，而且代码填空什么的完全不在话下，真的写的很开心，只是复习有些慢。

最后，它的复习方法，基于我课上并没有怎么听课的基础，我采用的是先简单的东西看PPT，看不懂的知识点看网课，以及梳理代码是如何写出来的也是看网课，听完至少可以写出七八分。问题在于，开始很晚，有畏难情绪，不想看易走神，所以拖到最后，学习内容分布不均，对学习时间错误估计。

告诉我们，复习较难的课程时，理应早早规划，多留时间，找到自己喜欢的复习方式不要全部一铺盖的听网课，记不住课，要自己动手记笔记，写代码。

最后的最后，就是我可能也还是喜欢计算机这个专业的吧，在看那些复杂而又优雅精妙的代码实现时，我还是会很开心很开心的啊，自己写出来的时候，真的很开心。
